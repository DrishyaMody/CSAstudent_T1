Polymorphism basics

```java
// Base class
class Animal {
    public void speak() {
        System.out.println("Animal speaks");
    }
}

// Subclass
class Dog extends Animal {
    @Override
    public void speak() {
        System.out.println("Dog barks");
    }
}

// Using polymorphism
Animal pet = new Dog();
pet.speak();  // Output: Dog barks
```
